안정 정렬 알고리즘

삽입(Insertion) 정렬
최선 시간복잡도 : O(n)
평균 시간복잡도 : O(n^2)
최악 시간복잡도 : O(n^2)

버블(Bubble) 정렬
최선 시간복잡도 : O(n)
평균 시간복잡도 : O(n^2)
최악 시간복잡도 : O(n^2)

병합(Merge) 정렬
최선 시간복잡도 : O(nlogn)
평균 시간복잡도 : O(nlogn)
최악 시간복잡도 : O(nlogn)
메모리 : n


불안정 정렬 알고리즘

선택(Selection) 정렬
최선 시간복잡도 : O(n^2)
평균 시간복잡도 : O(n^2)
최악 시간복잡도 : O(n^2)

퀵(Quick) 정렬
최선 시간복잡도 : O(nlogn)
평균 시간복잡도 : O(nlogn)
최악 시간복잡도 : O(n^2)
메모리 : logn ~ n

힙(Heap) 정렬
최선 시간복잡도 : O(nlogn)
평균 시간복잡도 : O(nlogn)
최악 시간복잡도 : O(nlogn)

계수(Counting) 정렬
최선 시간복잡도 : O(N + K)
평균 시간복잡도 : O(N + K)
최악 시간복잡도 : O(N + K)

인트로(Intro) 정렬
퀵 정렬에 힙 정렬을 더한 형태
기본적으로 퀵정렬을 하지만 재귀가 깊어지는 경우 힙 정렬을 사용한다.
최선 시간복잡도 : O(nlogn)
평균 시간복잡도 : O(nlogn)
최악 시간복잡도 : O(nlogn)
메모리 : logn
